# gox

A compile-time augmentation layer for Go. Adds sum types, exhaustive pattern matching, and full-stack contracts — all compiled down to valid, idiomatic Go.

## What it does

`gox` reads `.gox` files and outputs `.go` files. No runtime dependencies. No reflection. Just safer, more expressive Go.

### Sum Types

```
sumtype OrderState {
    Pending
    Paid { amount: float64, paidAt: time.Time }
    Shipped { trackingNumber: string }
    Cancelled { reason: string }
}
```

Compiles to a sealed Go interface with per-variant structs.

### Exhaustive Matching

```
match state : OrderState {
    Pending => fmt.Println("waiting")
    Paid(amount, paidAt) => processPaid(amount, paidAt)
    Shipped(trackingNumber) => ship(trackingNumber)
    Cancelled(reason) => cancel(reason)
}
```

Missing a variant is a **compile-time error**. No more forgotten cases.

### Contracts

```
contract CreateUser {
    input {
        email: string     @required @email
        password: string  @required @minlen(8)
        name: string      @optional
    }
    output {
        id: string
        createdAt: time.Time
    }
    errors {
        EmailTaken 409
        WeakPassword 400
    }
    route POST /api/users
}
```

Generates: input/output structs, validation function, typed error variants, HTTP handler, and typed client.

## Install

```bash
go install github.com/liamparker17/gox@latest
```

Or build from source:

```bash
git clone https://github.com/liamparker17/gox.git
cd gox
go build -o gox .
```

## Usage

```bash
# Compile .gox files to .go
gox compile myfile.gox
gox compile ./src/

# Check for errors without generating code
gox check myfile.gox
```

## Example

Given `orders.gox`:

```
package orders

import "time"
import "fmt"

sumtype OrderState {
    Pending
    Paid { amount: float64, paidAt: time.Time }
    Shipped { trackingNumber: string }
    Cancelled { reason: string }
}

func HandleOrder(state OrderState) {
    match state : OrderState {
        Pending => fmt.Println("waiting")
        Paid(amount, paidAt) => fmt.Printf("paid %f at %v", amount, paidAt)
        Shipped(trackingNumber) => fmt.Println("shipped:", trackingNumber)
        Cancelled(reason) => fmt.Println("cancelled:", reason)
    }
}
```

Run `gox compile orders.gox` to produce `orders_gen.go`:

```go
// Code generated by gox from orders.gox. DO NOT EDIT.
package orders

import (
    "fmt"
    "time"
)

type OrderState interface {
    isOrderState()
}

type OrderStatePending struct{}
func (OrderStatePending) isOrderState() {}

type OrderStatePaid struct {
    Amount float64
    PaidAt time.Time
}
func (OrderStatePaid) isOrderState() {}

// ... plus type switch with destructuring
```

## Annotations

| Annotation | Effect |
|---|---|
| `@required` | Zero-value check (empty string, zero number, nil, `.IsZero()` for time) |
| `@optional` | `omitempty` JSON tag, no required check |
| `@email` | Must contain `@` |
| `@minlen(N)` | Minimum string length |
| `@maxlen(N)` | Maximum string length |
| `@min(N)` | Minimum numeric value |
| `@max(N)` | Maximum numeric value |

## Design Principles

- Output must pass `go build`
- Generated code is readable and idiomatic
- No runtime dependencies
- Compile-time safety over runtime checks
- Exhaustiveness enforced by default (`//gox:ignore exhaustive` to opt out)

## Requirements

- Go 1.24+
